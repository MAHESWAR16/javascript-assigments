‚úÖ return
Where it is used?
‚û°Ô∏è Inside a function
What it does?
‚û°Ô∏è Stops the entire function
‚û°Ô∏è Sends a value back (optional)
‚û°Ô∏è No further code in that function will run

return  does TWO things:
Stops the function immediately (no more code runs after return)
Sends a value OUT of the function

When return runs, the function STOPS immediately.

Example:

function test() {
    console.log("A");
    return 10;
    console.log("B"); // this will NOT run
}

test();


Output:
A
Because:
"A" prints
return 10 runs ‚Üí function stops
"B" does NOT run
