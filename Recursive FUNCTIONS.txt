RECURSIVE FUNCTION DEFINATION : recursive means calling  the same function again.
üëâ Function calls do NOT erase the old function. They pause it, execute the new one, then return back to continue.
‚≠ê Rule of Recursion (VERY IMPORTANT)
Every recursive function needs two things:
1Ô∏è‚É£ Base condition (stop condition)
‚Üí Stops recursion
2Ô∏è‚É£ Recursive call
‚Üí Calls the function again

‚úÖ Program: Print "hello" 2 times using recursion
function printHello(count) 
{

    // 1. STOP when count becomes 0
    if (count === 0)
     {
        return;
    }

    // 2. Print
    console.log("hello");

    // 3. Call function again (reduce count)
    printHello(count - 1);
}

printHello(2);

‚ö° A function call PAUSES the current function.
‚ö° When the called function finishes, the paused function continues from the next line.

That‚Äôs what ‚Äúgoing back‚Äù means.
üö¶ Let‚Äôs show it step-by-step with a simple diagram
Your code:

function abc(i) {
  console.log("i =", i);

  if (i >= 3) {
    return;
  }

  abc(i + 1);
  console.log("after call", i);
}

abc(1);

üî• VISUAL EXPLANATION (THIS WILL MAKE 100% SENSE)
Imagine each function call is a page in a notebook.

‚û§ STEP 1: abc(1) starts
i = 1
Then it sees:
abc(2)

üìå abc(1) stops HERE and waits.
It creates a new page for abc(2).

‚û§ STEP 2: abc(2) starts
i = 2
Then it sees:
abc(3)

üìå abc(2) stops HERE and waits.
It creates another new page for abc(3).

‚û§ STEP 3: abc(3) starts
i = 3
Now:
if (i >= 3) return;

‚úî abc(3) returns
‚úî abc(3) page is finished

üìå Now the program returns to the previous paused page ‚Üí abc(2)
‚≠ê THIS IS THE ‚ÄúGO BACK‚Äù MOMENT
üìç You paused abc(2) at:
abc(3);   // <-- paused here


When abc(3) finishes, abc(2) continues from the next line:
console.log("after call 2");
And then abc(2) finishes.

‚≠ê Now it ‚Äúgoes back‚Äù to abc(1)
Because abc(1) was paused at:
abc(2);   // paused here
Now abc(2) has finished.
So abc(1) continues:
console.log("after call 1");


Then abc(1) finishes.
üéâ THAT'S ALL
To summarize:

abc(1) calls abc(2)  ‚Üí abc(1) pauses
abc(2) calls abc(3)  ‚Üí abc(2) pauses
abc(3) returns       ‚Üí go back to abc(2)
abc(2) returns       ‚Üí go back to abc(1)
abc(1) returns       ‚Üí program ends
